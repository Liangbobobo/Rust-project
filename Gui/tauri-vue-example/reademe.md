                                      1. 确定版本差异： 检查了 package.json、Cargo.toml 和 Cargo.lock                    文件，以了解声明版本和实际版本之间的差异。                                   2. 寻找兼容的 Beta 版本： 使用 npm view @tauri-apps/cli versions 和 npm view       @tauri-apps/api versions 命令查找了 CLI 和 API 包的共同且存在的 Beta            版本。最终选择了 2.0.0-beta.13 作为兼容版本。                                3. 更新 `package.json`： 修改了 app/package.json，将 @tauri-apps/api 和            @tauri-apps/cli 都明确设置为 2.0.0-beta.13。                                 4. 更新`Cargo.toml`： 修改了 app/src-tauri/Cargo.toml，将 tauri 和                tauri-build 都明确设置为 2.0.0-beta.13。                                     5. 重新运行`npm install`： 在 app 目录下执行 npm install，更新了                  package-lock.json 并安装了指定的 NPM 包版本。                                6. 添加缺失的 Rust 依赖： 在 app/src-tauri 目录下使用 cargo add                    tauri-plugin-opener 命令，将缺失的 tauri-plugin-opener crate 添加到             Cargo.toml 并解决了其版本。                                                  7. 重新运行`cargo build`： 在 app/src-tauri 目录下执行 cargo                      build，构建成功。                                                            8. 验证应用程序启动： 运行 npm run tauri dev，确认应用程序可以正常启动。
